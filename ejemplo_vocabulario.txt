# =============================================================
#  VocabRecall – Ejemplo de vocabulario estructurado
#  Formato:  anverso ; reverso
# =============================================================
#
#  SEPARADORES SOPORTADOS (se detectan automáticamente):
#
#    ;       punto y coma       →  das Haus ; the house
#    |       barra vertical     →  der Hund | the dog
#    TAB     tabulador          →  die Katze	the cat
#    -       guion con espacios →  groß - big
#    –       guion largo        →  schnell – fast
#
#  REGLAS:
#    • Una línea = una tarjeta (anverso → reverso)
#    • Las líneas que empiezan con # son comentarios (se ignoran)
#    • Las líneas vacías se ignoran
#    • Si el anverso incluye artículo (der/die/das), se extrae
#      automáticamente y se muestra por separado en la flashcard
#
# =============================================================
